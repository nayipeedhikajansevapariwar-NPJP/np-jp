API SPEC (serverless functions under /api)
- GET /api/reviews => returns list from Supabase
- POST /api/reviews => body { name, text, rating } validate text required; save { id, name, text, rating, date }
- GET /api/contacts => returns contacts (admin only)
- POST /api/contacts => body { name, email, location, message } validate required; save { id, name, email, location, message, date }
- GET /api/volunteers => admin only (header x-admin-key = ADMIN_KEY)
- POST /api/volunteers => body { name, email, phone, city, skills, availability, consent } validate required; save record
- POST /api/create-order => body { amount } if RAZORPAY configured create order, else return 501
